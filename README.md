# 📅 Dates Reminder Bot

Семейный бот для управления памятными датами. Node.js + TypeScript + MongoDB (FerretDB).

## Возможности

| Команда    | Описание                            |
| ---------- | ----------------------------------- |
| `/start`   | Приветствие и список команд         |
| `/dates`   | 📋 Список доступных дат             |
| `/add`     | ➕ Добавить новую дату              |
| `/edit`    | ✏️ Редактировать дату               |
| `/delete`  | 🗑 Удалить дату                     |
| `/private` | 👑 Управление ролями (только админ) |
| `/cancel`  | ❌ Отменить текущее действие        |

---

## Ролевая модель

| Роль                 | Видит                       | Создаёт            | Редактирует/Удаляет |
| -------------------- | --------------------------- | ------------------ | ------------------- |
| 👑 Админ             | Все даты                    | Любые              | Все                 |
| ⭐ Привилегированный | Все публичные + свои личные | Личные и публичные | Публичные + свои    |
| 👤 Обычный           | Все публичные               | Только публичные   | Публичные           |

---

## Уведомления

Все уведомления отправляются в **10:00** по часовому поясу из `.env`:

- За **3 дня** до даты
- **Накануне** (за 1 день)
- **В день** события

Публичные даты → в групповой чат.
Личные даты привилегированных → лично в ЛС.

---

## Типы дат

- 🔔 **Напоминание** — общее напоминание
- 🎂 **День рождения** — день рождения
- 📅 **Памятная дата** — памятное событие (по умолчанию)

## Периоды повторения

- **Один раз** — разовое событие
- **Каждый месяц** — ежемесячное повторение
- **Каждый год** — ежегодное повторение

## Быстрый старт

```bash
npm install
cp .env.example .env
# заполните .env
npm run build
pm2 start dist/index.js --name dates-bot
pm2 save
```

### .env

```
BOT_TOKEN=           # Токен от @BotFather
MONGODB_URI=mongodb://localhost:27017/dates_bot
ADMIN_TELEGRAM_ID=   # Ваш Telegram ID
GROUP_CHAT_ID=       # ID группового чата (-100xxxxxxxxxx)
TIMEZONE=Europe/Moscow
```

> ⚠️ Бот должен быть добавлен в групповой чат. Для получения GROUP_CHAT_ID добавьте бота в чат и напишите `/start` — ID появится в логах.

---

## Получить GROUP_CHAT_ID

Добавьте бота в группу, напишите там `/start`. В логах PM2 появится:

```
pm2 logs dates-bot
```

Найдите строку с `chat.id` — это и есть GROUP_CHAT_ID.
